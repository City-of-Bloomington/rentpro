Welcome to Rental System


Please Select one of the options in the left panel as follows:
  • To do a fast and simplified search for registrations where owner information or address details are not important try 'Simple Search'.
  • To do an extensive search for certain registrations, select the 'Extensive Search' option
  • To do a simplified search for certain owners, select the 'Search Owners' option
  • To check or search for certain address click on "Check Address".
  • To run reports with statistcis, choose Statistcs Report
  • For more information about this system, click on the Help option.
  • When you are done, please Logout from this System and close the session information.
